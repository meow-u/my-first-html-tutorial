CSS 구조

        선택자: 요소를 선택
        중괄호: 선언들을 작성한다.
        선언: 속성:값 세미콜론(;) 구분된다.
        예시: h1 {color:blue; font-size:16px;}

COLORS를 표현하는 다양한 방법

        색상명: blue
        hex : #0000ff;
        rgb : rgb(0,0,255);
        rgba : rgba(0,0,255,0.2); (a는 투명도임 20%투명도)
        hsl:hsl(240,100%,100%);
        hsla:hsla(240,100%,100%,0.5);
